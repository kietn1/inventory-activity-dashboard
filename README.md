# Inventory Activity Dashboard

This is a Streamlit dashboard for Item Activity Report Excel files.

## Main Features

- Upload Excel file online
- Auto-clean Item Activity Report
- SKU-level inventory summary
- Shortage forecast
- Risk level calculation
- Inbound vs outbound trend
- SKU transaction lookup
- Exception report
- Export CSV results

## Main Tabs

1. Overview
2. Shortage Forecast
3. Movement Trend
4. SKU Detail
5. Exceptions

## How to Use

1. Open the Streamlit app
2. Upload Item Activity Report Excel file
3. Review KPI cards
4. Check Shortage Forecast
5. Download CSV if needed
